Stuff I've worked on

Rupeebee

RupeeBee is a financial awareness and fraud prevention platform built for the SAFE Hackathon 2025 by Punjab & Sind Bank. Designed to make money matters simple and safe, the project was selected for presentation at Global FinTech Fest 2025.

Live Demo

Farmer Sahayak

AI Agricultural Advisor is a multilingual AI system for farmers in India, offering voice, image, and text-based agricultural guidance. Built with Next.js, it supports 11 Indian languages, plant disease detection, weather-aware advice, and a no-login PWA experience.

Github Live Demo

Road Anomaly Detection

Road Anomaly Detection is a computer vision project using YOLOv8 to detect road defects such as cracks and potholes. It includes custom datasets, trained models, evaluation results, and demo applications.

Github Live Demo

dumbAF

Project dumbAF by Team CollabDoor is a centralized academic resource hub for students. It brings together PYQs, syllabus, books, study materials, and essential resources in one place.

Github Live Demo

notacoder

notacoder is a beginner-friendly coding playground built with Django, designed to help users learn Python through bite-sized, hands-on challenges. It offers a low-stress, no-login experience with points, progress tracking, and playful feedback to make learning feel approachable and fun.

Github

Disposable Chatroom

A real-time disposable chatroom built with Svelte and GunDB. Users can join instantly, message in real time, delete their own messages and dispose of the room with a single click.

Github Live Demo

EV Battery Monitor

Quilles-Pulse is an IoT-based battery health monitoring system for EV charging using ESP32, MQTT, and Node-RED. It tracks battery and environmental data in real time with alerts and live dashboard visualization.
